\subsection*{inet\_ntoa: convert a 32 bit IPv4 address to a string}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] IPv4 address as a string
\end{description}

\subsubsection*{Description}

The \subroutine{inet_ntoa} subroutine takes a 32 bit, IPv4, address
and converts it to a string suitable for humans.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
