\subsection*{msgdsize: return the size data in a STREAMS message block }

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] The size of the data contained in the
  message block.
\end{description}

\subsubsection*{Description}

The \subroutine{msgdsize} subroutine returns the size of the data
contained in a message block stucture.  Message blocks are specific to
the STREAMS system.

\subsubsection*{Constraints}

The \subroutine{msdgsize} subroutine is only available on the Illumos
operating system.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
