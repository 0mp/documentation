\subsection*{toupper: convert a string to upper case}

\subsubsection*{Subroutine prototype}

\begin{verbatim}
char * toupper(const char *string);
\end{verbatim}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{arg0}] Pointer to the string
\item[\registerop{rd}] A string with only upper case letters
\end{description}

\subsubsection*{Description}

The \subroutine{toupper} subroutine converts the characters of the
string supplied as its only argument into upper case and returns the
resulting string.

\subsubsection*{Pseudocode}

\begin{verbatim}
string = stack[0].value
destination = scratch_space

for i = 0 ... len(string):
    c = string[i]
    if c is lowercase:
        c = uppercase(c)
    destination[i] = c

scratch_space += len(string)
%rd = destination
\end{verbatim}

\subsubsection*{Constraints}

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
