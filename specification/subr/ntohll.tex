\subsection*{ntohll: convert a long long (64 bit) value from network
  to host byte order}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] long long (64 bit) value in host byte order
\item[\registerop{arg0}] long long (64 bit) value in network byte order
\end{description}

\subsubsection*{Description}

The \subroutine{ntohll} subroutine takes a long long (64 bit) value as
its only argument and returns that value in host byte order.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
