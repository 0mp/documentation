\subsection*{cleanpath: return the cleaned up pathname}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] A pointer to the scratch space string containing the
	cleaned up pathname.
\end{description}

\subsubsection*{Description}

The \subroutine{cleanpath} subroutine takes a single string argument, containing
a path, and returns a pointer to a string containing the cleaned up pathname.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
