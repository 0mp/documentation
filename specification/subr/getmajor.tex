\subsection*{getmajor: return major device number}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] Major device number
\end{description}

\subsubsection*{Description}

The \subroutine{getmajor} subroutine returns the major device number
from a device structure supplied as the first argument.

\subsubsection*{Constraints}

The \subroutine{getmajor} subroutine is only available on Illumos and
systems derivice from OpenSolaris.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
