\label{subr:ntohl}
\subsection*{ntohl: convert long (32 bit) value from network to host
  byte order}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] value in host byte order
\item[\registerop{arg0}] value in network byte order
\end{description}

\subsubsection*{Description}

The \subroutine{ntohl} routine takes a 32 bit value as its only
argument and returns that value in host byte order.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
