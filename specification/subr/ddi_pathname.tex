\subsection*{ddi\_pathname: look up device driver by name}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{arg0}] Pointer to a device node.
\item[\registerop{arg1}] Device minor number.
\item[\registerop{rd}] Path within the /devices tree.
\end{description}

\subsubsection*{Description}

The \subroutine{ddi_pathname} subroutine returns a string describing
the device driver that implements a device in the system.

\subsubsection*{Constraints}

The \subroutine{ddi_pathname} subroutine is only available on Illumos and
systems derivice from OpenSolaris.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
