\subsection*{copyout: copy data from a buffer into process address space}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] void
\item[\registerop{arg0}] Pointer to buffer
\item[\registerop{arg1}] Pointer to memory
\item[\registerop{arg2}] Length
\end{description}

\subsubsection*{Description}

The \subroutine{copyout} subroutine copies data from a buffer supplied
by the caller into a process's address space.

\subsubsection*{Failure modes}

This subroutine has no run-time failure modes beyond its constraints.
