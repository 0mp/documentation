\subsection*{LDGAA: load a value from a hash map}

\subsubsection*{Format}

\textrm{LDGAA key, \%rd}

\begin{center}
\begin{bytefield}[endianness=big,bitformatting=\scriptsize]{32}
\bitheader{0,7,8,23,24,31} \\
\bitbox{8}{0x34}
\bitbox{16}{key}
\bitbox{8}{rd}
\end{bytefield}
\end{center}

\subsubsection*{Description}

The \instruction{ldgaa} instruction loads a value into the
\registerop{rd} register based on a key.  The key is used to lookup
the value in a hash map data structure.

\subsubsection*{Pseudocode}

\begin{verbatim}
%rd = map[key]
\end{verbatim}

\subsubsection*{Failure modes}

This instruction has no run-time failure modes beyond its constraints.
