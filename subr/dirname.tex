\subsection*{dirname: return the directory component of a pathname}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] A string pointing to the directory component of
  a pathname.
\end{description}

\subsubsection*{Description}

The \subroutine{dirname} subroutine returns a string containing the
directory component of a pathname, without the terminating filename.
