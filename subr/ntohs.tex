\subsection*{ntohs: convert a short (16 bit) value from network to
  host byte order}

\subsubsection*{Calling convention}

\begin{description}
\item[\registerop{rd}] short (16 bit) value in host byte order
\end{description}

\subsubsection*{Description}

The \subroutine{ntohs} subroutine takes a short (16 bit) value as its
only argument and returns the same value in host byte order.
